# Paper Grain Texture Required

This file needs to be created/sourced:

**File:** `public/paper-grain.png`

**Requirements:**
- Paper grain texture image
- Should be a seamless/repeating pattern
- Will be applied at 15% opacity with multiply blend mode
- Used as background texture overlay

**Note:** The CSS is already configured to use this file. Once added, it will automatically appear as a subtle texture overlay on all pages.
